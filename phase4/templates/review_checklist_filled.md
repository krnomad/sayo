# Review Checklist (Filled Template)

- [ ] 변경 이유가 `changed_files`와 정합한가?
- [ ] `related_graphyfi_paths` 근거가 최신(stale 아님)인가?
- [ ] 영향을 받는 architecture 문서가 누락되지 않았는가?
- [ ] 보안/컴플라이언스 관련 항목 검토가 완료되었는가?
- [ ] 인터페이스 계약 변경 시 승인자가 지정되었는가?
- [ ] 자동 생성 결과를 source-of-truth에 직접 반영하지 않았는가?
- [ ] 승인 전 상태(`pending` 또는 `blocked_manual_sync`)가 기록되었는가?
- [ ] trace metadata(4종 commit hash)가 기록되었는가?
